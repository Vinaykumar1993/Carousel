## Getting Started

### For Development


```
npm install

npm run dev // For Dev Server
```

### For Production
```
npm run build // For generating webpack output with Production environment
```
### For SSR
```
npm run ssr // For generating webpack output with Server Sider Rendering
```

then localhost:3000
